## Build up of the Badge ID

* the first 13 characters are the registrar ID (It belongs to a person that is a custom client developer or a org thats maintaining a custom client)
* the last 13 characters are the badge ID (This is freely chosen by the Registrar)

## How to get a registrar ID

you need to be a developer/maintainer of a custom client that supports this custom badge system.
Only the person thats hold the repo for the client can get a registrar ID and there can only be **one** registrar per custom client.
